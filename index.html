Kebijakan Privasi Elvoya

Terakhir diperbarui: 4 Juli 2026
Elvoya dirancang untuk menyimpan kenangan pribadimu. Semua data momen — foto, audio, catatan, dan lokasi — disimpan hanya di perangkatmu dan tidak pernah dikirim ke server kami.

1. Data yang Kami Kumpulkan

Elvoya mengakses data berikut di perangkatmu:

  • Kamera dan foto — untuk mengambil atau memilih foto sebagai bagian dari momen.
  • Mikrofon — untuk merekam memo suara (fitur Echo).
  • Lokasi — untuk menandai lokasi momen di peta. Lokasi hanya diambil saat kamu membuat momen dan disimpan lokal di perangkat.
  • Penyimpanan — untuk menyimpan foto, audio, dan sketsa yang kamu buat.

2. Cara Kami Menggunakan Data

Semua data yang dikumpulkan digunakan semata-mata untuk menjalankan fitur aplikasi:

  • Menampilkan momen di Timeline dan peta.
  • Memutar ulang memo suara.
  • Mengelompokkan momen berdasarkan lokasi dan mood.

Kami tidak mengumpulkan, mengunggah, atau menjual data pribadimu kepada pihak ketiga manapun.

3. Penyimpanan dan Backup Data

Seluruh data momen disimpan secara lokal di perangkatmu menggunakan database internal aplikasi. Kami tidak memiliki server cloud sendiri dan tidak pernah mengakses datamu.

Elvoya menggunakan fitur Android Auto Backup bawaan Google untuk mencadangkan data aplikasi (database momen, sketsa, dan memo suara) ke akun Google Drive milikmu. Backup ini dikelola sepenuhnya oleh Google dan hanya dapat diakses oleh kamu. Foto yang kamu ambil disimpan di Galeri perangkat dan dicadangkan sesuai pengaturan galeri yang kamu miliki (misalnya Google Photos).

Kamu dapat menonaktifkan backup ini melalui menu Pengaturan → Profil → Backup Otomatis di dalam aplikasi, atau melalui pengaturan Google Backup di perangkatmu.

4. Layanan Pihak Ketiga

Elvoya menggunakan layanan Google berikut: AdMob (iklan), Firebase Analytics (statistik penggunaan anonim), dan Firebase Crashlytics (laporan crash teknis). Semua layanan ini dikelola oleh Google sesuai Kebijakan Privasi Google.

5. Izin Aplikasi

Elvoya meminta izin berikut:

  • CAMERA — mengambil foto untuk momen.
  • RECORD_AUDIO — merekam memo suara Echo.
  • ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION — menandai lokasi momen.
  • READ_MEDIA_IMAGES — memilih foto dari galeri.
  • POST_NOTIFICATIONS — mengirim pengingat momen (opsional).
  • INTERNET — memuat peta dan iklan AdMob.

Semua izin hanya digunakan untuk tujuan yang disebutkan di atas.

6. Data Anak-Anak

Elvoya tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak dengan sengaja mengumpulkan data dari anak-anak.

7. Perubahan Kebijakan

Kami dapat memperbarui kebijakan privasi ini dari waktu ke waktu. Perubahan akan ditandai dengan tanggal pembaruan di bagian atas halaman ini. Penggunaan aplikasi setelah perubahan berarti kamu menyetujui kebijakan yang diperbarui.
